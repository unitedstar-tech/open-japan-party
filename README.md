# Open Japan
Open Japanは、Cosmopolitan libertarianismに立脚した社会を実現するための新しい政党です。
## Cosmopolitan libertarianismとは
私達が重視するCosmopolitan libertarianismは、以下のような価値観です。
* **個人の自由の重視**: リバタリアニズムに基づき、国家の介入を最小限に抑え、個人の自由な選択と行動を最大限に尊重します。
* **世界市民としての平等な扱い**: コスモポリタニズムに基づき、人種、民族、国家、地域に関わらず、すべての個人を人類という共同体の一員として平等に扱います。
* **開かれた国境と自由貿易**: 私達は、国境を越えた人(移民)、モノ、資本の移動に対する障壁を取り除くことを強く推進します。
* **非介入主義的な外交政策**: 他国の政治への軍事的・政治的介入を避ける「不介入主義」を堅持します。
* **普遍的な道徳的配慮**: 特定の国家や集団の利益よりも、世界全体の個人の福祉や権利を普遍的な基準で考慮します。
私達は、国籍や場所に関わらず、すべての個人は平等な自由を持つべきであり、政府は彼らの移動や交流を制限すべきではないと考えます。
